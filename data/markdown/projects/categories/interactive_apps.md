# Human-Computer Interaction Systems, Real-Time Chat, and Game Prototypes

### FlashChat - Student chat application for Campus
Cross-platform mobile app built with Flutter and Firebase for students to swap quick updates, ask questions, and hang out in topic-based rooms, without the clutter of public social media.

### Tech Stack
Flutter SDK, Dart, Firebase Core, Cloud Firestore, Cross-platform

### Math Garden - Interactive Toddler App
A browser-based addition game for toddlers and early learners where they write the answer by hand, a small MLP neural network recognizes the digit in real time, and a colorful garden grows or shrinks based on their answers.

### Tech Stack
Python, Tensorflow, JavaScript, HTML/CSS, TensorFlow.js

### Mood Synergizer - Emotion Aware Music Player
Desktop application that reads a user's facial expression from the webcam, detects their dominant emotion, and instantly launches a curated playlist that matches the mood. Designed as a playful way for students to unwind after work or studying.

### Tech Stack
Python, Tkinter, OpenCV, DeepFace, Pillow, Pygame

### Unity Gameplay Suite
A compact suite of four Unity projects that highlight core gameplay programming, game loops, and clean scene management. Focus on readable C#, reusable components, and clear player feedback. Each project ships with a short gameplay demo.

### Tech Stack
Unity, C#, Animator, NavMesh, ScriptableObjects, UI Toolkit
